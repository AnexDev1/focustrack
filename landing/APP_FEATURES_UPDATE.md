# FocusTrack App And Dashboard Update

## Product Summary

FocusTrack is now a cross-device screen time and focus analytics product instead of a desktop-only tracker. The desktop app remains the main analytics hub, while the Android mobile app acts as a companion that records mobile usage, syncs it to desktop over the local network, and enforces app limits directly on the phone.

This means users can track how they spend time across desktop windows and mobile apps in one product, while keeping the data local and under their control.

## New Mobile App Capabilities

### 1. Native Android Screen Time Tracking

The mobile app reads Android usage events and turns them into app sessions for the current day. It captures:

- app name
- session start time
- session end time
- session duration

This gives FocusTrack a true mobile screen time layer instead of a manual estimate.

### 2. Real-Time Mobile Dashboard

The mobile dashboard now shows a live screen time card that updates continuously, along with:

- quick daily stats
- app usage distribution
- top apps list
- recent mobile sessions

The goal is to make the phone experience feel like a real tracker, not just a sync client.

### 3. Desktop Sync Over Local Network

The Android app can sync its usage sessions to the desktop app over the same network. This makes the desktop app the main place to review combined activity while the phone keeps collecting mobile usage in the background.

### 4. App Limits And Locking

Users can set daily limits for selected Android apps. When a limited app reaches its allowed time, FocusTrack blocks access with a dedicated lock screen instead of a dismissible message. The lock remains in place until the user changes the limit inside FocusTrack.

This turns FocusTrack from a passive analytics app into an active self-control tool.

## Desktop Dashboard Changes Caused By The Mobile App

The desktop dashboard was expanded so it can work as a unified analytics surface for both desktop and mobile data.

### 1. Source Toggle In Charts And Stats

The dashboard now supports three source modes:

- Combined
- Desktop
- Mobile

This lets users answer different questions:

- Combined: how much time did I spend across all devices?
- Desktop: what did I do on my computer only?
- Mobile: what happened on my phone only?

### 2. Auto-Refresh After Sync

When the mobile app sends new sessions to the desktop app, the dashboard refreshes automatically. Users no longer need a hot restart or manual reload to see newly synced mobile activity.

### 3. Mobile Stats Inside Desktop

The desktop dashboard now includes mobile-aware summaries so the desktop app is no longer blind to phone usage. This makes the desktop experience a true command center for screen time analysis.

### 4. Unified Visual Analytics

The usage chart and summary cards now react to the selected source. This makes the same dashboard useful for:

- full cross-device review
- desktop-only focus analysis
- mobile-only habit review

## New Mobile Features Added Recently

### Live Screen Time Counter

The mobile screen time card updates live so users can see time increasing without needing a hot reload.

### Background Sync Flow

The mobile app can keep feeding fresh mobile usage into the shared analytics flow, which helps desktop stay up to date.

### Persistent App Blocking

Once an app reaches its limit, FocusTrack opens a dedicated blocking screen that prevents normal back-navigation into the blocked app. This makes the limit system far more reliable.

### Focus-Oriented Limits Experience

The mobile app is no longer only about reporting. It now supports behavior change by letting users set limits and immediately feel the enforcement on-device.
